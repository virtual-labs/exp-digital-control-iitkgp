Instructions

  1. Enter the values of the Mechanical system parameters. Note: Default values are already given.

  2. Click on 'State Space Model' button to get the continuous state space form of the system.

  3. Click on ' Discrete State Space Model' button to get the discrete state space form.

  4. Click on 'Check for Controllabilty' button to get check the controllability of the system.

  5. Click on the 'Rank' or 'Determinant' or "inference' buttons to get the the Controllability test information.

  6. Clicking on 'Design' dropdown-menu to select the type of the design.

  7. Click on the 'Run' button to run the selected design.

  8. Click on ' Enter the pole Location' button to enter the desired pole values (in s domain) for state feedback method.

  9. Click on ' Desing' button to get the gain values for the state feedback method.

  10. Click on 'Clear' button to enter the new values of quantities.
